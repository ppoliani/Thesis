Entity - Method

Properties
Name Value
Data Model Physical
Documentation

A method can be anything that calculated the carbon emissions. For example, an extrapolation method for calculating the annual carbon emissions for an individual, based on partial set of trips that she has added. This information can be used to track down the provenance of the annual individual carbon emissions.
