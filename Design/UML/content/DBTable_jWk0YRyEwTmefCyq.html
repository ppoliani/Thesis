DDLClauses A trip might have several parts. For example, a trip from home to work might consist of two sub-trips; a trip with ferry form home and a trip with a bus to work. This table represents the parts of the trip.

Columns Summary
Column Name Data Type PK / FK Nullable Documentation
id integer(10) PK No Record id
tripID integer(10) FK No The id of the trip that it is part of.
startAddressID integer(10) FK No The id of starting address
endAddress integer(10) FK No The id of end address
transportMeanId integer(10) FK No The id of transport mean that was used
step integer(10) No The step of the trip. In other words the stage of the trip that this leg represents
time time(7) No The time when the trip was made
