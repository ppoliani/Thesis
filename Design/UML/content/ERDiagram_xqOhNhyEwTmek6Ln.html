Database Tables
Name Documentation
 Users : Entity 		Table with the system's users
 Groups : Entity 		Any group that contains a number of people, such as a research group, an academic unit or the whole university.
 University : Entity 		Table describing a university
 GroupMembers : Entity 		It contains user's and the groups they are member of
 AcademicUnits : Entity 		Academic units of a faculty.
 Faculties : Entity 		Faculties of a university
 Campuses : Entity 		University campuses
 Addresses : Entity 		A table containing details about addresses.
 Trips : Entity 		Table containing the trips made by students and the members of the university staff
 TripLegCarbonEmissions : Entity 		Table containing the estimated carbon emissions of each trip.
 Cars : Entity 		Table containing some information about cars. The original data set is located in http://carfueldata.direct.gov.uk/
 transportMeansUsedByUsers : Entity 		Table containing all the transport means that were used by all the users.
 EFSources : Entity 		Table containing the sources for emission factors, which are used for in the carbon emission calculation formula.
 userLocations : Entity 		Table containing user's addresses
 CalculationMethods : Entity 		Table containing the different methods for calculating carbon emissions
 UniversityCampuses : Entity 		Table containing the campuses of a university
